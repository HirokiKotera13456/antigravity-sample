## 🌌 Concept

Cafe Eternity は、単なるカフェではなく「体験としての空間」を目指しています。  
Marni や Prada のようなハイブランドの美学にインスパイアされた、余白を大胆に活かしたデザインと、タイポグラフィの美しさを追求しました。

- **Antigravity**: 「無重力」をテーマにした微細な浮遊アニメーション。
- **Minimalism**: 視覚的ノイズを削ぎ落とした、静寂を感じるレイアウト。
- **Artistic Selection**: 一杯のコーヒーを一つの作品として提示。

## 🛠 Technology Stack

本プロジェクトは、最新のフロントエンド技術を使用して構築されています。

- **Core**: React 19 + Vite 8
- **Language**: TypeScript (Strict Mode)
- **Styling**: Tailwind CSS v4 (Modern CSS Engine)
- **Animation**: Framer Motion (Fluid Interactivity)
- **Icons**: Lucide React
- **Design System**: カスタムテーマ変数による一貫したブランドカラー管理

## 🎨 Design System

ブランドのアイデンティティを守るための厳選されたカラーパレット：

| 要素 | カラーコード | 用途 |
| :--- | :--- | :--- |
| **Background** | `#F8F7F2` | 温かみのあるオフホワイト |
| **Text** | `#1A1A1A` | 深い墨色 |
| **Accent** | `#800020` | 落ち着いたボルドー |
| **Highlight** | `#FFDB58` | 洗練されたマスタード |

## 🚀 Getting Started

プロジェクトをローカル環境で起動する方法：

### 1. 依存関係のインストール
```bash
npm install
```

### 2. 開発サーバーの起動
```bash
npm run dev
```
サーバーが起動したら、 [http://localhost:5173](http://localhost:5173) をブラウザで開いてください。

### 3. ビルド
```bash
npm run build
```
